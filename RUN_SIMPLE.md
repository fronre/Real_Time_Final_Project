# 🚀 تشغيل سريع ومضمون

## طريقة التشغيل الصحيحة

### الخطوة 1: تشغيل السيرفر
```bash
cd server
python main_simple.py
```

### الخطوة 2: فتح الواجهة
افتح المتصفح واذهب إلى: `frontend/index_simple.html`

**أو مباشرة:**
```bash
# افتح الملف مباشرة
start frontend\index_simple.html
```

## 🔧 إذا لم يعمل

### تأكد من:
1. **بايثون يعمل**: `python --version`
2. **السيرفر شغال**: `python main_simple.py`
3. **الواجهة مفتوحة**: `index_simple.html`
