# 🚀 Real-Time Trading System - Easy Setup

## للإعداد السهل والمشاركة مع الأصدقاء

---

## 📋 خيارين للتشغيل (Two Options to Run)

### 🔥 الخيار 1: دوكر (Docker) - الأسهل للمشاركة
```bash
# 1. ثبّت دوكر من: https://www.docker.com/products/docker-desktop/
# 2. شغّل الأمر:
docker-compose up --build
# 3. افتح: http://localhost:3000
```

### 🛠️ الخيار 2: يدوي (Manual) - بدون دوكر
```bash
# 1. ثبّت GCC و Python
# 2. Windows: شغّل build.bat ثم start.bat
# 3. Linux/Mac: شغّل ./build.sh ثم ./start.sh
# 4. افتح: http://localhost:8000
```

## 📱 ما ستراه في النظام

- **10 أصول رقمية**: بيتكوين، إيثيريوم، عملات ألعاب، اشتراكات، NFTs
- **تداول تلقائي**: إشارات شراء وبيع تلقائية
- **توقيت دقيق**: بالميكروثانية
- **رسوم بيانية مباشرة**: أسعار وتداولات حية
- **خوارزمية EDF**: جدولة المهام المتقدمة

## 🔧 حل المشاكل السريع

### إذا لم يعمل دوكر:
```bash
# نظف وحاول مرة أخرى
docker-compose down
docker system prune -f
docker-compose up --build
```

### إذا لم يعمل التشغيل اليدوي:
```bash
# Windows
# تأكد من تثبيت MinGW و Python
# شغّل: build.bat ثم start.bat

# Linux/Mac
# تأكد من تثبيت gcc و python
# شغّل: ./build.sh ثم ./start.sh
```

## 🎮 نصائح للمشاركة

### أفضل طريقة للمشاركة:
1. **استخدم دوكر** - لا يحتاج أي إعداد تقني
2. **أرسل رابط دوكر** + الأمرين للتشغيل
3. **أخبرهم ينتظروا 2-3 دقائق** للبناء الأول

### رسالة جاهزة للأصدقاء:
```
مرحبا! 🚀

عندي نظام تداول أصول رقمية حقيقي مع خوارزمية EDF

التشغيل:
1. ثبّت دوكر من هنا: https://www.docker.com/products/docker-desktop/
2. حمّل الملفات التي أرسلتها لك
3. افتح طرفية في مجلد المشروع
4. شغّل: docker-compose up --build
5. انتظر 2-3 دقائق ثم افتح: http://localhost:3000

ستظهر لوحة تداول مباشرة مع بيتكوين وإيثيريوم وغيرها! 🎉

إذا واجهت أي مشكلة، أخبرني وسأساعدك.
```
